# Arkaprava Ray — Portfolio Website

Pure static HTML. Works by simply double-clicking any .html file.
No build step, no server, no installation needed.

## ✅ How to open locally
1. Extract this ZIP anywhere on your computer
2. Double-click index.html — it opens in your browser, fully styled
3. All pages, images, CSS and JS are self-contained in this folder

## ⚙️ Two things to configure before publishing online

### 1. Contact Form — Formspree (free)
1. Sign up at https://formspree.io
2. Create a new form → copy the Form ID (e.g. xabcdefg)
3. Open contact.html → find YOUR_FORMSPREE_ID → replace with your ID

### 2. Visitor Analytics — GoatCounter (free)
1. Sign up at https://www.goatcounter.com → choose a site code (e.g. arkapravaray)
2. In ALL .html files, find YOUR_SITE_CODE → replace with your code
3. View your dashboard at https://arkapravaray.goatcounter.com

## 🚀 GitHub Pages (free hosting)
1. Create a GitHub repo named <username>.github.io
2. Upload ALL files from this folder into the repo root
3. Settings → Pages → Source: main branch, / (root) → Save
4. Live at: https://<username>.github.io

## 📱 How to test on mobile before publishing
- Chrome: F12 → phone icon (top-left) → pick iPhone or Android preset
- Firefox: F12 → Responsive Design Mode
- Or just share the GitHub Pages URL and open on your phone

## Features
- Hamburger menu on mobile (≤768px), full nav on desktop
- Full-screen image lightbox on research pages
- Publication highlight animation when linked from research page
- Contact form via Formspree
- Visitor analytics via GoatCounter
